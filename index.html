Modular

Sheweny is modular and you can use only the components (structures/managers) you need.

Modern

Sheweny is built with modern technologies.

Customizable

Sheweny is fully customizable and you can change all parameters/options.

TypeScript

Sheweny is built with TypeScript and exports all types.
