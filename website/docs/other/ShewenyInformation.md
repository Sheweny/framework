# ShewenyInformation

Put an information in console in development mode.

## Constuctor

```js [Javascript CJS]
new ShewenyInformation(client, 'Hello word !');
```

| Name    | Type                                        | Description |
| ------- | ------------------------------------------- | ----------- |
| client  | [ShewenyClient](../client/ShewenyClient.md) | The client  |
| message | [String]                                    | The message |
